Запуск: python NT_collector_file_dev.py "C:\folder\folder"
Собирает данные о файле (хэш файла,имя файла, размерб дату изменения, путь) и заливает в БД 
с помощью вызова хранимой процедуры "_add_fileversion".
Чтобы проверить работоспособность без БД, нужно закомментировать строки: 51,52,53,59,61,62,63.
